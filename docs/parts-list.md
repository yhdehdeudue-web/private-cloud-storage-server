# Parts List

## Required parts

- Raspberry Pi 5, 4GB or better
- Raspberry Pi active cooling fan
- MicroSD card
- 27W USB-C Raspberry Pi power supply
- NVMe HAT adapter board or external USB SSD adapter
- M.2 NVMe SSD or external SSD
- Ethernet or Wi-Fi connection

## Optional parts

- Bumper case
- Argon Neo 5 case or another all-in-one Pi 5 NVMe case
- Extra backup drive
- Ethernet cable

## Notes

The YouTube tutorial used a Raspberry Pi 5 with an NVMe SSD. I may start by planning and documenting the project first, then build the physical setup once I have the hardware.
